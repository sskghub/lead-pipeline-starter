# Contributing

Thanks for improving this template.

1. Fork the repo and create a branch.
2. Install dev hooks: `pre-commit install`
3. Run checks: `pre-commit run --all-files`
4. Run a local secret scan before pushing: `trufflehog filesystem .`
5. Open a PR with a short description of the change.

Please do not commit real API keys, tokens, or service account JSON.
